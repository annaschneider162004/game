# Minecraft Schematic City Generator

Tool tự động tạo file `.schematic` cho phép xây dựng những thành phố ảo trên Minecraft với sự hỗ trợ của nhiều nhân vật bot.

## Tính năng

- 🏗️ Tạo tự động file `.schematic` cho các cấu trúc thành phố
- 🤖 Hỗ trợ nhiều nhân vật bot xây dựng song song
- 🏢 Sinh tòa nhà, đường phố, công viên tự động
- ⚙️ Cấu hình linh hoạt qua file config
- 📦 Hỗ trợ định dạng NBT schematic
- 🎨 Tuỳ chỉnh vật liệu và style xây dựng

## Quick Start

```bash
pip install -r requirements.txt
python examples/generate_city.py
```
